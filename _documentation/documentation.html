01. Setting up the template

A. Editing and configuration

Aside from the regular editing of the index.php, where you adjust the content to your needs, first you'll have to adjust the configuration settings in inc-settings.php.

B. PHP Support

In order for the template to function properly, your server needs to be setup to interpret PHP scripts, but that's a given with most of today's hosting companies.

C. Installation

Setting up the template is a very easy job! Basically, all you have to do is to move the contents of the downloaded archive to your FTP account, with or without the documentation folder.

02. Required and optional configurations

In order to adjust the template's configuration you will need to open the inc-settings.php file located in the root folder of the archive with your text/code editor of choice (Notepad works just fine, if nothing else is available).

A. Required configurations values:

  • WEBMASTER_EMAIL

Make sure you fill in your real and valid email address, otherwise the contact form will not work.

B. Optional configurations values:

  • COLOR_SCHEME (Accepted values: empty, default, light, dark, red, green, cyan, purple, wood or baroque)
  • SM_BLOG_URL (Please include "http://")
  • SM_TWITTER_ID
  • SM_FACEBOOK_ID
  • SM_LINKEDIN_ID
  • SM_FLICKR_ID
  • SM_YOUTUBE_ID
  • SM_YAHOO_MESSENGER_ID
  • SM_SKYPE_ID

All settings with names ending in "_ID" should not contain full URL paths, but only the USERNAME you have set.
An example of username that you could pick up for Flickr is: http://www.flickr.com/people/qbkl/. The filled in setting would look like this:

define ('SM_FLICKR_ID', 'qbkl');

You can either fill in all of the social media settings or none. These are all options and the social media menu displayed in the template will be generated automatically based on the settings that you have filled in.

03. Editing the template

In its predefined form, the "Bold" CV / resume template contains the most common information and utilities found in resumes:

  • Name, job title, photo, introductory message and contact information (with social media profiles);
  • Personal and professional profile;
  • Technical Skills (Aptitudes);
  • Work experience / history (Current and previous jobs);
  • Education;
  • Client list and work samples;
  • Contact form;
  • Print template and a PDF generation module.

In order to easily create and add new sections to your CV/resume, we have prepared a documentation page that provides you with the 4 sample sections and the code used to generate them. Click here to load this sample page.

Also, before heading in to editing the template or creating new sections yourself, here are some things that might come in handy:

<div class="ruler"></div> Use this piece of code whenever you want to create a new row of columns within a section. Best exemplified in section 4 of the sample sections page.
mt0, mt10, mt20, mt30 Using these classes on your resume's HTML elements will set those elements' top margin to 0, 10, 20 or 30 pixels. When working with more content within a section or starting your column's content with a heading you'll find it useful to negate the default top margin of that heading like this <H3 class="mt0">
Column classes
  • <div class="column"> - creates a 1/3 content column.
  • <div class="column titlecolumn"> - creates a 1/3 title column with a double border at top for design purposes. Only use as first column in a row to preserve style. Inside it, use <h2 class="sectiontitle"> to define your section titles.
  • <div class="largecolumn"> - creates a 2/3 content column and should only be used after a <div class="column"> or a <div class="column titlecolumn">
  • A .largecolumn can act like a container for a 2-column structure as sampled in section 4 of the sample sections page.
  • When creating the last column in a row (3rd in a 3 column structure, or 2nd if contained within a .largecolumm) always add the class .lastcolumn to negate its right margin, like this: <div class="column lastcolumn">
Technical skill levels Setting up your technical skill level is pretty easy. The visual representation of a skill level is obtained with a piece of code like this: <div class="skillbar" title="Average"><div class="skillbarfill skill40">Average</div></div>. You will notice the skill40 class applied. This means the yellow bar will fill 40% of the ruler. You have at your disposal 10 such classes to use, in 10% steps: skill10, skill20, skill30, and so on until skill100.
Also, make sure you set the title of the .skillbar div to a meaningful value (used for the tooltip) and the content of the .skillbarfill div to the same value (used for the print template).
